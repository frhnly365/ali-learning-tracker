⚙️ Cara Deploy di Notion

  1. Buat page baru → pilih “Import” → “Markdown” atau copy-paste HTML ke block “Embed”.
  2. Untuk setiap section, buat sub-page dengan database sesuai tabel di atas.
  3. Paste HTML di embed, dan sesuaikan view (Calendar, Board, Timeline).
  4. Tambahkan Template Buttons pada masing-masing database jika perlu.
